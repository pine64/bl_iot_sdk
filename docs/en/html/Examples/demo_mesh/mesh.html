14. Mesh¶

14.1. Overview¶

This example mainly introduces how to use ble mesh.

14.2. Use steps¶

  • Prepare the phone to install the APP bl_mesh_app.apk

  • Compile customer_app/bl602_demo_eventthe project and download the project;

  • Run Mesh related commands in the serial port:

    #stack_ble

    #blemesh_init

    #blemesh_pb 2 1

  • Open the App, select and click Add in the Network column

  • In the scanning interface, select the device to be added

  • Click the IDENTIFY option in the interface

  • Click the PROVISION option in the interface

  • Select No OOB in the pop-up interface and click ok

  • In the pop-up interface, click ok

  • The interface is automatically transferred to the Network column, click on the successfully connected device

  • Select the Elements option in the interface and click its drop-down button

  • Select the Generic on off Server option in the interface

  • Select the BIND KEY option in the interface

  • Click Application key 1 in the interface

  • Click the ONOFF option in the interface to control the LED switch

  • The following information is seen in the serial port to indicate that the LED is successfully controlled

  • Click the SUBSCRIBE option in the interface,

  • Select the Create a new group to subscribe option in the interface,

  • Follow the same method to add another device node. After the addition is successful, in the Groups option, click the successfully created group

  • In the group, select ON/OFF, and print log in the serial port to prompt LED switch information, indicating that the mesh networking is successful
