Developer Environment¶

此目录介绍编译或下载等工具的使用。

编译¶

用户可以使用Linux平台或Windows平台进行编译，具体使用方法参考 Linux环境编译 , Windows环境编译

下载¶

用户可以使用BL自主开发的工具 BLFlashEnv 进行下载，具体使用方法参考 BLFlashEnv

Debug¶

用户可以在 Eclipse + OpenOCD 和 Freedom Studio + OpenOCD 两种方式中选择一种方式进行debug。具体调试方法参考 Eclipse 和 Freedom Studio 。
