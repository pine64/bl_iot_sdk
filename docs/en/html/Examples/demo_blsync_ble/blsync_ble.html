15. BLSYNC-BLE¶

15.1. Overview¶

This example mainly introduces how to use ble for wifi configuration.

15.2. APP use steps¶

  • Compile customer_app/sdk_app_ble_syncthe project and download the firmware engineering;

  • When the firmware is powered on, it will automatically start ble broadcast, and wait for the mobile APP to connect to the distribution network, as shown below;

  • Open the mobile APP to search for Bluetooth devices, and search for the device name "BL602-BLE-DEV";

  • After clicking the connected device, click Scan in the APP, and after a few seconds, the APP will display the list of wifi devices scanned by the development board;

  • Users can connect to the wifi that needs network configuration through the scanned device list;

  • When the user confirms that the network configuration is completed, the network configuration function does not need to be used, and can be turned off by using the "blsync_ble_stop" command.

15.3. Steps to use WeChat Mini Program¶

  • Compile customer_app/sdk_app_ble_syncthe project and download the firmware engineering;

  • When the firmware is powered on, it will automatically start ble broadcast, and wait for the mobile APP to connect to the distribution network, as shown below;

  • Open the WeChat applet to search for Bluetooth devices, and search for the device name "BL602-BLE-DEV";

  • Click "BL602-BLE-DEV" to connect the device. After connecting to the BLE of the device, the BLE service will be obtained. Click the first service, and then select "Write Notification";

  • Click "click to configure network" in the applet, the applet will echo the obtained wifi list;

  • Users can connect to the wifi that needs network configuration through the scanned device list, and click the name of the wifi that needs to be connected;

  • Then enter the wifi password in the input box and click "send password" to connect to wifi;

  • Click the "Get Status" button in the applet to get the current connection status of wifi;

  • Currently connected to wifi, it will display "Connected" and pop up information such as the ip address of the board;

  • Click the "Disconnect wifi" button to disconnect the wifi, and click the "Get Status" button again to get the current wifi disconnected;

  • When the user confirms that the network configuration is completed, the network configuration function does not need to be used, and can be turned off by using the "blsync_ble_stop" command.
