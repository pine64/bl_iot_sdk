5. UART_ioctl¶

5.1. Overview¶

This example mainly introduces how to set multiple modes of UART1.

5.2. Related configuration¶

  • Configuration Reference uart_echoconfiguration.

  • Steps for usage

  • The board gpio3and gpio4and GNDrespectively USB转TTLthe serial lines TXD, RXD, GNDare respectively connected;

  • Compile customer_app/sdk_app_uart_ctlthe project and download the project;

  • Open a serial terminal window A (115200 baud rate, used to receive and send uart messages), and open another serial terminal window B (baud rate 2000000, used to print log). For example, use the configuration in case1 in the application implementation, enter 123456789abcdefin window A, and you can see the received data in window A, and the number of received data is printed in window B.
